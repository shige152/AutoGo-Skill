# Opencv 模块 API 文档

提供基于 OpenCV 的图像处理功能。由于 OpenCV 方法数量太多，剩余的方法全部参照官方文档。

## API 函数

### FindImage

在指定区域内查找匹配的图片模板，支持透明图像（图像的四个角颜色相同且透明像素数量占总像素数的 30% 以上）处理。返回找到的图片左上角坐标，如果未找到则返回 (-1, -1)。

**参数：**
- x1, y1 {int}: 区域左上角的坐标
- x2, y2 {int}: 区域右下角的坐标。当 x2 或 y2 为 0 时，表示使用图像的最大宽度或高度
- template {*[]byte}: 模板图片的字节数组指针，表示要在区域内查找的图片
- isGray {bool}: 是否将图像转换为灰度图进行匹配，提升匹配速度和鲁棒性
- scalingFactor {float32}: 缩放比例，应用于模板图片和截取的待匹配区域，确保模板能够适应不同分辨率的图像
  - 0.5 表示缩小为原图的 50%
  - 2.0 表示放大为原图的 200%
  
**应用场景：**
模板图像在原分辨率（例如 720x1280）下截取的，但需要在更高（例如 1080x1920）上查找时，通过调整 scalingFactor 为 1.5 使模板图像可以在不同的分辨率上正常查找。

例如：
- scalingFactor 为 2.0 时，模板会放大 200%，适用于将 540x960 的模板匹配到 1080x1920 的图像中
- scalingFactor 为 0.5 时，模板会缩小 50%，适用于将 540x960 的模板匹配到 270x480 的图像中
- scalingFactor 为 1.0，则模板和截取区域保持原尺寸，不进行缩放

- sim {float32}: 相似度阈值，取值范围为 0.1 到 1.0，值越高表示匹配要求越精确
- displayId {int}: 屏幕 ID

**返回值：**
- x {int}: 找到的图片左上角 x 坐标，未找到返回 -1
- y {int}: 找到的图片左上角 y 坐标，未找到返回 -1

```go
x, y := opencv.FindImage(0, 0, 1920, 1080, &templateBytes, true, 1.0, 0.8, 0)
if x != -1 && y != -1 {
    fmt.Printf("模板匹配成功，坐标为：%d, %d\n", x, y)
} else {
    fmt.Println("未找到匹配的模板。")
}
```

### FindImageAll

在指定区域内查找匹配的图片模板，支持透明图像处理，返回所有符合条件的坐标。

**参数：**
- x1, y1 {int}: 区域左上角的坐标
- x2, y2 {int}: 区域右下角的坐标。当 x2 或 y2 为 0 时，表示使用图像的最大宽度或高度
- template {*[]byte}: 模板图片的字节数组指针，表示要在区域内查找的图片
- isGray {bool}: 是否将图像转换为灰度图进行匹配，提升匹配速度和鲁棒性
- scalingFactor {float32}: 缩放比例，应用于模板图片和截取的待匹配区域，确保模板能够适应不同分辨率的图像
- sim {float32}: 相似度阈值，取值范围为 0.1 到 1.0，值越高表示匹配要求越精确
- displayId {int}: 屏幕 ID

**返回值：**
- points {[]Point}: 所有匹配到的坐标点数组

```go
points := opencv.FindImageAll(0, 0, 1920, 1080, &templateBytes, false, 1.0, 0.8, 0)
for _, point := range points {
    fmt.Printf("找到匹配，坐标：%d, %d\n", point.X, point.Y)
}
```
